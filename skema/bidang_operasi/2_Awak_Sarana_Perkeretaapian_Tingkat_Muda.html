LSP PT Kereta Api Indonesia (Persero)
Awak Sarana Perkeretaapian Tingkat Muda

Kode Unit Judul Unit Kompetensi
KAI.OO.001.00 Menerapkan Peraturan Perundang-undangan Serta Standar Keselamatan dan Kesehatan Kerja di tempat kerja
KAI.OP.001.00 Melakukan Kegiatan Langsir
KAI.OP.002.00 Mengoperasikan Sarana Kereta Api
KAI.OP.012.00 Mengoperasikan Sarana Kereta Api saat Kondisi Tidak Normal
Unduh Panduan
Download
Biaya
Rp604.600

Persyaratan Dasar
  1. Pendidikan minimal SLTA.
  2. Memiliki Sertifikat Awak Sarana Perkeretaapian Tingkat Pertama.
  3. Pengalaman kerja telah bertugas sebagai Awak Sarana Perkeretaapian Tingkat Pertama selama paling sedikit 1 (satu) tahun atau 2000 (dua ribu) jam kerja yang dibuktikan dengan melampirkan buku catatan jam kerja (logbook).
  4. Lulus pendidikan dan Pelatihan teknis Awak Sarana Perkeretaapian Tingkat Muda sesuai dengan jenis sarana Perkeretaapian berpenggerak yang dioperasikan.
  5. Sehat jasmani dan rohani.
  6. Tidak buta warna.
  7. Tinggi badan minimal 160 cm.
